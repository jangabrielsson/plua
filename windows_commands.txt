Corrected Windows Commands for Clearing Interpreter Cache
========================================================

1. First check if debug code exists in the installed file:
python -c "import plua.interpreter; import inspect; print('DEBUG' in inspect.getsource(plua.interpreter.LuaInterpreter.initialize))"

2. PowerShell command to clear cache (run in PowerShell):
Get-ChildItem -Path "$env:LOCALAPPDATA\Packages" -Recurse -Filter "*interpreter*.pyc" | Remove-Item -Force

3. Command Prompt alternative (run in cmd):
for /r "%LOCALAPPDATA%\Packages" %f in (*interpreter*.pyc) do del "%f"

4. Clear Python module cache:
python -c "import sys; [sys.modules.pop(k, None) for k in list(sys.modules.keys()) if 'interpreter' in k]; print('Module cache cleared')"

5. Test plua:
plua

Notes:
- The original command had incorrect wildcard syntax
- Use PowerShell version if possible (more reliable)
- Run commands one at a time
